# Style Templates per Thai Sub-Genre

Each template is a ready-to-paste Styles field. Pick the sub-genre that matches the user's request. Adjust BPM, mood, and instrument list to taste — but keep the structure (Genre → Sub-genre → Mood → Instruments → Vocal → Production → BPM → Thai cue).

All templates avoid artist names per the no-artist-names rule in `suno-fields-reference.md`.

## How to combine code-switching cues

When the user wants Thai+English mix, append one of these to the end of the Styles field:

- `Thai vocals primary with English chorus echoes` — for Thai verses + English in `(parens)` echoes
- `bilingual Thai and English vocals, alternating sections` — for half-half
- `Thai verses with one English bridge` — for one English section only

This helps Suno avoid drifting between languages mid-line.
